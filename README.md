# Wiener Linien Ticket System: Human-Centric Security Analysis

[cite_start]Bu proje, teknik olarak güvenli kabul edilen bir sistemin (Viyana Ulaşım Ağı - Wiener Linien), kognitif önyargılar ve sosyal mühendislik yöntemleriyle nasıl manipüle edilebileceğine dair bir "Proof of Concept" (PoC) çalışmasıdır[cite: 1172, 1175].

## Proje Özeti
[cite_start]Sistem teknik olarak güçlü şifrelenmiş QR kodlar kullansa da, kontrolörlerin (insan faktörü) yoğun çalışma temposunda QR kod taramak yerine görsel ipuçlarına (hareketli saat, renk paleti) öncelik verdiği gözlemlenmiştir[cite: 1176, 1213, 1221].

## Teknik Yaklaşım
- [cite_start]**Teknoloji:** JavaScript ve React Native (Snack Expo) kullanılarak uygulamanın birebir görsel kopyası geliştirilmiştir[cite: 1259, 1260].
- [cite_start]**Siber Güvenlik Vektörü:** "Social Engineering" ve "Cognitive Biases" (Confirmation, Anchoring, Normalcy Bias)[cite: 1241, 1242, 1270].
- [cite_start]**Analiz:** Boston ve Londra gibi şehirlerdeki eski RFID tabanlı sistemlerin teknik açıkları ile Viyana'daki insan odaklı sistem zafiyetleri karşılaştırılmıştır[cite: 1192, 1193, 1198].

## Önemli Not
[cite_start]Bu çalışma tamamen akademik amaçlı bir güvenlik analizidir[cite: 1238, 1266]. [cite_start]Uygulamanın tam kaynak kodları, kötüye kullanımı önlemek amacıyla burada paylaşılmamaktadır; sadece analiz raporu sunulmuştur[cite: 1237, 1282].
